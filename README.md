# سوشيال سيفر — SocialSaver 📥

أداة مجانية مفتوحة المصدر لتحميل الصور والفيديوهات **بجودتها الأصلية** من منصات التواصل الاجتماعي، تعمل بالكامل داخل المتصفح (Bookmarklet) بدون سيرفرات وبدون إعلانات.

**الموقع:** https://ashrafcontact6086-web.github.io/Instagram-photo-downloader/

## المنصات المدعومة

| المنصة | الدعم |
|---|---|
| إنستغرام | صور + فيديو + ريلز (محرّك مخصص) |
| تيك توك | فيديو + منشورات الصور (محرّك مخصص) |
| إكس (تويتر) | صور بدقة `orig` + فيديو (محرّك مخصص) |
| فيسبوك | فيديو HD + صور (محرّك مخصص) |
| ثريدز | صور + فيديو (محرّك مخصص) |
| بنترست | صور أصلية + فيديو (محرّك مخصص) |
| أي موقع آخر | وضع عام ذكي (og tags + JSON-LD + عناصر الصفحة) |

> يوتيوب غير مدعوم عمداً: روابط الفيديو مشفّرة والصوت منفصل عن الصورة، فلا يمكن تحميله من المتصفح بجودة موثوقة.

## طريقة الاستخدام

1. افتح الموقع واسحب زر **«سوشيال سيفر»** إلى شريط المفضلة.
2. افتح صفحة أي منشور (وإن كان فيديو شغّله لثوانٍ).
3. اضغط الزر من المفضلة → تظهر لوحة بكل الوسائط → حمّل فردياً أو دفعة واحدة أو كملف ZIP.

## الخصوصية

لا يوجد أي سيرفر وسيط: الأداة تقرأ روابط الوسائط الأصلية من بيانات صفحة المنصة نفسها ويتم التحميل مباشرة بين متصفحك والمنصة. لا يُرفع أو يُسجَّل أي شيء.

## تنبيه قانوني

استخدم الأداة فقط للوسائط التي يحق لك تحميلها، واحترم حقوق النشر وشروط استخدام كل منصة.
